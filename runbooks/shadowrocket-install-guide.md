# Shadowrocket（小火箭）购买与安装教程

## 准备工作

- iPhone 或 iPad
- 一个海外 Apple ID，例如美区、日区、港区
- App Store 礼品卡或可用的海外支付方式
- 你的订阅链接

## 步骤 1：切换 App Store 到海外区

1. 打开设置，点击顶部 Apple ID。
2. 进入“媒体与购买项目”，退出登录。
3. 打开 App Store，点击右上角头像，退出当前账号。
4. 用海外 Apple ID 登录 App Store。

注意：这里只切换 App Store 购买账号，不需要退出 iCloud。

## 步骤 2：购买 Shadowrocket

1. 在 App Store 搜索 `Shadowrocket`。
2. 确认开发者和图标无误后点击购买。
3. 使用礼品卡余额或海外支付方式付款。

Shadowrocket 通常约 $2.99，价格以 App Store 实际显示为准。

## 步骤 3：导入订阅

1. 打开 Shadowrocket。
2. 点击右上角 `+`。
3. 类型选择 `Subscribe`。
4. 粘贴客服发给你的订阅链接。
5. 点击完成，返回首页。
6. 首页应看到 Reality 和 Trojan 两个节点。
7. 点击节点右侧信息按钮测试延迟，选择延迟较低的节点。
8. 打开顶部代理开关。

## 步骤 4：测试访问

依次测试：

- TikTok
- X / Twitter
- WhatsApp Web
- Facebook

如果页面可以正常加载，视频和图片正常显示，说明配置完成。

## 日常使用

- 平时优先使用 Reality 节点。
- Reality 连不上时，切换到 Trojan 节点。
- 两个节点都连不上时，先在 Shadowrocket 里刷新订阅，再重试。

## 常见问题

### 装不上 Shadowrocket

确认 App Store 已切换到海外区。国内区 App Store 通常搜索不到或无法购买。

### 订阅链接打不开

联系服务方重新生成订阅链接。不要把订阅链接转发给无关人员。

### 打开代理后仍访问不了

按顺序处理：

1. 刷新订阅。
2. 切换 Reality / Trojan 节点。
3. 关闭 Wi-Fi 后用蜂窝网络测试。
4. 联系服务方检查节点状态。
